You can use the following shortcodes in your listing template. WordPress shortcodes will not work here.

Available Shortcodes

[[product_title]]
product title

[[product_content]]
product main description

[[product_excerpt]] or
[[product_excerpt_nl2br]] or
[[product_additional_content]]
additional product description

[[product_price]]
product price

[[product_sku]]
product SKU

[[product_gallery]]
New product gallery with thumbnails and without active content

[[product_main_image]]
main product image as HTML tag

[[product_main_image_url]]
main product image as raw URL

[[product_thumbnails]]
clickable thumbnails for all images

[[additional_product_images]]
additional images with JS (deprecated)

[[img_1]] ... [[img_9]]
[[img_url_1]] ... [[img_url_9]]
single images or their urls

[[product_weight]]
[[product_height]]
[[product_width]]
[[product_length]]
product dimensions

[[attribute_Size]]
[[attribute_Brand]]
example for custom product attributes

[[product_variations]]
product variations as HTML table

[[product_category]]
main product category name

[[product_tags]]
comma-separated product tags

[[meta_custom-meta-field-name]]
custom meta values

[[widget_new_listings]]
[[widget_ending_listings]]
[[widget_related_listings]]
[[widget_featured_listings]]
Note: These legacy widgets are deprecated since eBay banned active content in 2017.
